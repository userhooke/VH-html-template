### NPM scripts:

run local browser sync server
```
npm run localhost
```

watch sass/scss
```
npm run sass:watch
npm run scss:watch
```

Ruby compile sass/scss
```
npm run sass
npm run scss
```

Stylelint
```
npm run stylelint
npm run stylelint-scss
```

Convert sass to scss or vice versa
```
npm run sass-scss
npm run scss-sass
```

Run pugjs watcher
```
npm run pug:watch
```

Compile pug files from Render folder
```
npm run pug
```
